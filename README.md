# MacGPT – Local AI Copilot

MacGPT is a privacy-first AI assistant built in Python using Retrieval-Augmented Generation (RAG) and a local LLM through Ollama.

## Features
- Fully local document ingestion and retrieval
- RAG pipeline with chunking, embeddings, and vector search
- Natural language to shell command execution
- Safety layer to block destructive commands
- Logging for prompts, commands, and outputs

## Tech Stack
- Python
- Ollama
- RAG
- Embeddings
- Vector search

## Highlights
- Handles 1000+ local documents
- <300ms retrieval latency
- 0 per-query inference cost with local models

## Purpose
Built to explore private, offline AI assistance for personal system workflows without relying on cloud APIs.
